# Asset Files Needed

## Favicons
Create two favicon files:

### favicon-light.png
- Size: 32x32 or 64x64 pixels
- Design: Green leaf or farming icon on light/transparent background
- Use for light mode

### favicon-dark.png
- Size: 32x32 or 64x64 pixels  
- Design: Same icon but optimized for dark backgrounds
- Use for dark mode

## Videos
Place in `/public/videos/`:
- `home.mp4` - Homepage background
- `about.mp4` - About page background

See `/public/videos/README.md` for specifications.

## Optional Images
- `/public/og-image.jpg` - Open Graph image (1200x630px)
- `/public/twitter-image.jpg` - Twitter card image (1200x600px)

## Quick Setup:
You can use free tools like:
- Favicon: https://favicon.io/
- Icons: https://www.flaticon.com/
- Images: https://www.canva.com/

Or use placeholder services during development:
- Favicons: Use emoji or simple SVG
- Videos: Skip temporarily (app works without them)
