# Video Assets

Place your video files here:

- `home.mp4` - Homepage background video (farming scenes, crops, fields)
- `about.mp4` - About page background video (farmers working, technology in agriculture)

## Recommended Specifications:
- Format: MP4 (H.264 codec)
- Resolution: 1920x1080 (Full HD) or higher
- Duration: 10-30 seconds (will loop)
- File size: < 5MB for optimal loading
- Frame rate: 24-30 fps

## Free Video Sources:
- Pexels: https://www.pexels.com/search/videos/farming/
- Pixabay: https://pixabay.com/videos/search/agriculture/
- Unsplash: https://unsplash.com/s/videos/farm
